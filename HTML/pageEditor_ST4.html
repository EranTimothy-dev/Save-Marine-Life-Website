'placeholder' attribute i used to specify a short hint regarding the expected value of the input field.
'name' attribute is used to facilitate identiication of data.


CSS properties used

&:nth-child(7) :Targets the 7th child of a parent element.
input[type="submit"]:hover Applies styles when a submit button is hovered over.
@media Used for media queries to apply styles based on screen size or device type.
font-family: Specifies the font to be used for text.
backface-visibility: hidden; Hides the backface of an element during 3D transformations.
transform: scale(1.2); Scales the element to 120% of its original size.

Challenges and Lessons Learned

Many challenges were faced when defining sections and aligning them for the contents of the user profile page to have the best viewing experience.We used grids to define the contents side by side uniformly across the webpage.Therefore, it was identified that using a grid system is the best solution for this challenge.
Also it was a great challenge to select contrasting colours while preserving the website theme colours.To overcome this challenge i had to read on colour contrasting theories and visit many webpages with similar colour pallets.Got a great understanding on colour contrasting theories and applications.

Compliance

Ensured the pages I created do not host or link to any illegal, offensive, or inappropriate content.
Ensured that the content is relevant to its purpose and does not promote misinformation.
Ensured accessibility compliance by including text alternatives for images,keyboard navigability and high contrast ratios for the webpages i created.
